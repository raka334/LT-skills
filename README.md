# LT Skills

Shared agent skills for QA automation workflows.

## Install

Install this skill repo with:

```sh
npx skills add raka334/LT-skills
```

Install the `kane-cli-qa-converter` skill explicitly with:

```sh
npx skills add raka334/LT-skills/kane-cli-qa-converter
```

## Available Skills

- `kane-cli-qa-converter`: Converts generated or standalone QA scripts into an existing repository's automation framework and custom test flow.
